# ld40

Ludum Dare 40 Entry - The more you have, the worse it is

# Controls

```
WASD     to move
Mouse    to look around
Shift    to run
Control  to sneak
F11      to toggle fullscreen
```

# Build

To build and run, say:

```
go get -u github.com/gonutz/ld40
cd %GOPATH%\src\github.com\gonutz\ld40
build.bat
ld40.exe
```
